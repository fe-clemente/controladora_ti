Padrão de cache
Cada módulo tem sua própria pasta cache/ com arquivos JSON locais. Isso evita chamadas desnecessárias à API do Google Sheets. O fluxo é:
Browser → Express → lerCache() → retorna JSON local
                              ↘ se vazio: lê Sheets → salva JSON → retorna
O botão Sincronizar força a re-leitura da planilha e atualiza o cache.

Nginx mínimo (se usar domínio próprio)
nginxserver {
    listen 80;
    server_name sistemas.divinofogao.com.br;

    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_cache_bypass $http_upgrade;
    }
}
